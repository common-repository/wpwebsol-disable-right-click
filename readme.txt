== Description ==
 WpWebSol Disable Right Click a very simple and light weight plugin that is specialy design to prevent users from copy text, view source and inspect elements by right clicking.

== Installation ==

1. Upload the entire "wpws-disable-right-click" zip file through Plugins > Add New > Uplaod Plugin .
2. Activate the plugin through the Plugins > Installed Plugins.


For Step by Step demonstrations with screenshots.
Please Visit https://wpwebsol.com/plugins/disable-right-click/
